# zig-delve-triangle-study

![screenshot](screenshot.png)

Tiny example of using [delve](https://github.com/Interrupt/delve-framework) as external library.

Tested with Zig 0.13.0.

## Run

```bash
# debug:
$ zig build run

# release:
$ zig build run --release=fast
```
